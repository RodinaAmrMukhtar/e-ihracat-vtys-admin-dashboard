Neumorphic pastel frontend update

What changed:
- Softer light neumorphic UI inspired by the reference screenshot
- Redesigned sidebar, topbar, dashboard hero, stat cards, tables, and forms
- Backend and SQL connection files kept compatible with the working setup

How to run:
1) Backend terminal:
   cd backend
   npm install
   npm run dev

2) Frontend terminal:
   cd frontend
   npm install
   npm run dev

3) Open:
   http://localhost:5173

If you already have node_modules installed locally, you can simply replace the frontend/src folder,
frontend/index.html, and frontend/package.json if needed.
